# Taste UI (Learning Project)

Taste UI is a personal project where I explore how to combine frontend development with basic AI/ML concepts.

The idea is to generate UI themes from images and text, while learning how things like color extraction and simple ML algorithms work in practice.

## Tech Stack

* Next.js
* TypeScript
* Tailwind CSS
* (Upcoming) FastAPI + Python

## What I'm Learning

* Building UI with reusable components
* Connecting frontend with backend APIs
* Understanding basic ML concepts through implementation (e.g. K-Means for color extraction)
* Learning by doing instead of focusing only on theory

## Notes

This is an early-stage learning project. The code will evolve over time as I experiment, refactor, and improve my understanding.
